# sNNake 2.0

A multiplayer snake game.

The number of players and cpus can be defined in SnakeGame.

The keybindings for snake players are as follow:

|||
|---|---|
Snake 1:    | arrow keys
Snake 2:    | WASD
|||

Game key bindings:

|||
|---|---|
Pause:      | space bar
Restart:    | ctrl + r
Close:      | ctrl + c
|||
